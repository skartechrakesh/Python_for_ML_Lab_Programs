# Python_for_ML_Lab_Programs
**Lab Experiments**
1.	Variables and Data Types: Accept two numbers, perform arithmetic, display results, and explore type conversion and memory management.
2.	Control Flow and Exception Handling: Write a program to find the maximum of three numbers with exception handling for invalid inputs.
3.	Functions and Lambda Expressions: Implement a factorial function and filter even numbers using a lambda function.
4.	Data Structures: Develop a student record system using dictionaries and tuples, and explore set operations.
5.	File Handling: Read a text file, count word frequency, and save results to a new file.
6.	Object-Oriented Programming (OOP): Create a Bank Account class with methods for deposit, withdrawal, and balance checks, with validation.
7.	NumPy and Pandas Basics: Create a NumPy array, convert it to a Pandas DataFrame, and calculate basic statistics.
8.	Data Wrangling: Import a CSV, clean data by handling missing values, and rename columns.
9.	Querying and Merging DataFrames: Perform inner, outer, and left joins on sales data DataFrames.
10.	Aggregations: Group a dataset by a categorical column and calculate aggregate metrics.
11.	Applying Functions to DataFrames: Apply custom functions to modify DataFrame values.
12.	Visualizing Data with Matplotlib: Create line graphs, bar charts, and pie charts with proper labels and legends.
13.	Advanced Pandas Plotting: Create scatter matrices, lag plots, and autocorrelation plots for time series data.
14.	Seaborn Visualizations: Create heatmaps, pair plots, and regression plots with customized formatting.
